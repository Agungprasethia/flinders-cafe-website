# Kerja Praktek - Program

Repository ini berisi program yang dikembangkan untuk keperluan Kerja Praktek.

## Struktur Project

```
Program/
├── Admin dashbooard/    # Admin Dashboard (React + Vite + Tailwind CSS)
├── tampilan utama/      # Tampilan Utama / Beranda (React + Vite)
└── README.md
```

## Tech Stack

- **Frontend**: React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS (Admin Dashboard), CSS (Tampilan Utama)

## Cara Menjalankan

### Admin Dashboard
```bash
cd "Admin dashbooard"
npm install
npm run dev
```

### Tampilan Utama
```bash
cd "tampilan utama"
npm install
npm run dev
```

## Catatan
- Pastikan sudah menginstall [Node.js](https://nodejs.org/) sebelum menjalankan project.
- Jalankan `npm install` terlebih dahulu untuk menginstall dependencies.
